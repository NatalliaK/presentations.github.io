HTML

HyperText Markup Language

HTML is used to create the markup of a web page

HTML allows the browser to understand which elements you have placed on the page and where

CSS

Cascading Style Sheets

CSS is used to style the HTML elements on the page

CSS allows you to change the appearance of HTML components

(color, size, font, and many others)

JavaScript

programming language

JavaScript is used to add interactivity to the web page

JavaScript is used to handle user actions on the page

JavaScript allows for the manipulation of the HTML and CSS
